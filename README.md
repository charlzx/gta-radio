# GTA Radio

A GTA-inspired radio web app with synchronized playback, station browsing, and station details.

![License](https://img.shields.io/badge/License-MIT-green)

## Features

- Synchronized playback
- Real-time now playing/track detection
- Station details page with metadata and tracklist
- Track deep-link playback from station details
- Search across stations
- Favorites and recently played
- Cross-game genre browsing
- Responsive UI (desktop/mobile)

## Quick Start

### **Prerequisites**
- Node.js 18+ and npm (or yarn/pnpm)
- Modern web browser with HTML5 Audio support
- Internet connection for external audio streams

### Installation
```bash
git clone https://github.com/charlzx/gta-radio.git
cd gta-radio

npm install

npm run dev
```

### Build
```bash
npm run build
npm run preview
```

## Contributing

We welcome contributions to make GTA Radio even better! Here's how you can help:

### **Getting Started**
1. **Fork** the repository on GitHub
2. **Clone** your fork: `git clone https://github.com/charlzx/gta-radio.git`
3. **Create** a feature branch: `git checkout -b feature/your-feature`
4. **Install** dependencies: `npm install`
5. **Start** development: `npm run dev`

### **Development Guidelines**
- Follow existing code style and component patterns
- Test your changes across different screen sizes
- Ensure audio synchronization works properly
- Add appropriate comments for complex logic
- Update documentation for new features

### **Submission Process**
1. **Commit** changes: `git commit -m 'feat: add amazing feature'`
2. **Push** to branch: `git push origin feature/amazing-feature`
3. **Create** a Pull Request with clear description
4. **Wait** for review and address any feedback
